# Finger Music

Небольшой прототип виртуальной гитары: приложение отслеживает руку через веб-камеру и проигрывает ноты, когда кончики пальцев пересекают нарисованные струны.

## Что изменено в текущей версии

- запуск перенесен в `main()`, поэтому модуль можно импортировать без побочных эффектов;
- звук синтезируется программно, внешние WAV-файлы не нужны;
- струны срабатывают только внутри активной области гитары;
- проигрывание происходит по пересечению струны пальцем, а не по простому удержанию рядом с линией.

## Требования

- Python 3
- веб-камера
- файл модели `hand_landmarker.task` в корне проекта

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Покажите руку в камеру и проводите кончиками пальцев поперек струн. Для выхода нажмите `q`.

## Проверка логики

```bash
python -m unittest test_main.py
```

## Стек

- OpenCV
- MediaPipe Tasks
